# Live Call App

A real-time video conferencing application built with Node.js and 100ms SDK.

## Features

- User Authentication
- Create Video Rooms
- List the rooms


## Tech Stack

- Node.js
- Express
- PostgreSQL
- 100ms SDK
- WebSocket
- JWT Authentication
